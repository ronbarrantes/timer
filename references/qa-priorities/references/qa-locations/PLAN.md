# QA Locations extension

The goal will be to create an extension that contains the needed functionality for QA locations
and a similar deployment strategy for a zip-ready browser extension

The features go as follow

- [x] The extension will have 2 fields a create button and a reset button
- [x] field 1 will be name locations and the other one will be named priorities
- [x] we will take fields similar to the format SS4:HV253.A and sort them the same way as the existing Golang implementation
- [x] we will add locations in the same way, they dont have to be sorted
- [x] I also want the extension to hold a similar settings options to generate the location arrangement as the golang project
- [x] It should be able to deploy as a zip file ready for distribution
- [x] the table should have the highlighted fields as the priorities, similar to how the golang project does it
